
   
# Micro:bit 十六進制轉十進制積木
輸入為十六進制字串，轉成十進制輸出

主要用於 V7RC 專家模式上解碼使用

代表值  1000  ->  收到值 HEX : 64

代表值  1010  ->  收到值 HEX : 65   

代表值  1500  ->  收到值 HEX : 96 

代表值  2000  ->  收到值 HEX : C8
